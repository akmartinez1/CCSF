CS 110B Programming Fundamentals: C++

CS 110B Programming Fundamentals: C++ (3)
Lec-3, conf-1, lab-3
Credit, Degree Applicable
PREREQ.: CS 110A.

This course covers programming fundamentals using the C++ language. Topics 
include pointers, arrays, structured elements, dynamic memory allocation, and 
Object Oriented programming. Assignments require planning, good coding 
practices, and documentation.
